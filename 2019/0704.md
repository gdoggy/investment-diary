# 2019年07月04日

今日市场略微高开，随后维持昨日趋势，震荡下行

纵观盘面无任何亮点，热点炸板率极高，可以看出当前的行情完全依赖消息面刺激

总体上投资意愿再度陷入低迷，赚钱效应极低

当前虽无利空消息，但场上草木皆兵，增量资金不足，上行乏力

从结果来看前期过于乐观了，导致判断出现失误

今天持仓个股跌幅较大，已经完全回吐前期涨幅，进入浮亏状态

等待本周最后一个交易日行情结束，必须降低仓位，严格控制风险
