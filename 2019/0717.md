# 2019年07月17日

近几天市场情绪有所缓和，盘面有涨有跌，多空总体平衡

相比上个周频繁的单边下行行情，本周明显场面上更活跃

今日沪指低开，随后高走冲击2950后回落，延续昨日趋势

白马股延续不佳表现，“中国国旅(SH:601888)”开盘跌逾3%，随后触底反弹

操作策略上仍保持按兵不动，等待指数大幅回调的短期抄底机会

另外近期新股发行较多，均提交了申购，碰碰运气吧

短期指数看跌到2800点左右



收盘后突发消息面地雷

> 美国总统特朗普又说可能对价值3250亿美元的中国输美商品加征关税

又一次加剧了盘面的恐慌情绪
