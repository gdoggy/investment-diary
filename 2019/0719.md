# 2019年07月19日

下周一科创板开闸，本周最后一个交易日，场面上士气给足

A股高开高走，国家队早早的便敲锣打鼓，呐喊助威

早盘券商领队，悉数热点跟进，沪指从2900直冲2940，随后触顶回落

三大指数盘中一度涨逾1%，亚太其他股市普涨，场面上挺喜庆，就是不知道能持续多久

策略上继续围观，没有急于进场，毕竟盘面已经普涨，抢筹成本过高，不宜追高

静待下周的行情，历史性时刻，会发生什么拭目以待
