# A股投资笔记



2018年底的时候读到一本《穷查理宝典》，使我对投资有了新的认识。与此同时，随着贸易摩擦的进展，A股估值降低至近年来低位。2019年初国内出台减税降费利好，贸易谈判、冲突缓和，机会来了。。

## 2019

### 1月 — 4月

1月初入市，直到4月初都是躺着盈利，毫不夸张，真的躺盈。

4月开始赚钱效应减弱，接下来更是像过山车一样赤鸡。
